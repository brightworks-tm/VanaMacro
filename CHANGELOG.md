# Changelog

All notable changes to VanaMacro will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2025-12-04

### Added
- `VanaMacro.vbs` - 管理者権限でコンソールなしで起動するスクリプト（推奨）
- README: 起動方法の説明を追加
- README: 初回起動時の注意（FFXIから取り込みが必要）を追加
- README: マクロの反映タイミング（ログアウト→コピー→ログイン）の説明を追加

### Removed
- `run_admin.bat` - VanaMacro.vbsで代替

## [1.1.0] - 2025-12-02

### Changed
- テーマ適用時のフォント処理を簡素化 (Gameテーマ固有のフォント切り替えロジックを削除)
- 検証用スクリプトとレポートの追加 (`verify_ttl.py`, `検証結果報告.md`)
- UIエディタとテーマ設定の微調整

## [1.0.0] - 2025-11-28

### Added
- 初回リリース
- Git バージョン管理の導入
- プロジェクト完成版としてリリース

### Changed
- ドキュメントの整備と最終調整

## [0.9.0] - 2025-11-26

### Added
- 英語版UI実装完了
- 多言語対応の最終調整
- UI要素の英語翻訳辞書追加

### Fixed
- 言語切り替え時の状態保持の改善
- 翻訳漏れの修正

## [0.8.0] - 2025-11-24

### Added
- 即時言語切り替え機能の実装
- 設定ダイアログからの言語変更に対応
- 言語設定の永続化（config.json）

### Changed
- UI言語切り替えをアプリ再起動不要に改善
- 翻訳辞書のリロード機能追加

## [0.7.0] - 2025-11-20

### Added
- SQLiteデータベースへの移行
- `autotrans.db` による定型文・アイテム辞書管理
- 多言語データベース構造の実装（日本語/英語対応）

### Changed
- バイナリファイルから SQLite への辞書フォーマット変更
- 辞書更新ツール `sync_auto_tables.py` の実装

### Improved
- 辞書検索パフォーマンスの向上
- データ保守性の改善

## [0.6.0] - 2025-11-15

### Added
- エクスポート機能の実装
- エクスポートセンター UI の追加
- FFXI USER フォルダへの直接コピー機能
- マニフェストファイルによる検証結果記録

### Changed
- 取り込み処理時の自動エクスポートバックアップ
- タイムスタンプ付きエクスポート履歴管理

## [0.5.0] - 2025-11-10

### Added
- シンタックスハイライト機能
- コマンド、ターゲット、定型文の色分け
- ジョブアビリティ、魔法、ウェポンスキルの識別
- 行エラーチェック（60バイト制限、Shift-JIS検証）

### Improved
- マクロ編集の視認性向上
- エラー検出の強化

## [0.4.0] - 2025-11-05

### Added
- 定型文検索・挿入機能
- カテゴリ別定型文リスト
- ひらがな/カタカナ/ローマ字検索対応
- pykakasi による読み変換

### Changed
- 定型文ダイアログの UI 改善
- 検索性能の最適化

## [0.3.0] - 2025-11-01

### Added
- メインUI実装
- Book/Set/マクロ選択機能
- 6行エディタと一括テキスト編集の双方向同期
- キャラクター管理機能
- テーマ切替機能（Base/Dark/Game）

### Changed
- データ保護機能の強化（未保存警告）
- 編集中内容のメモリ保持

## [0.2.0] - 2025-10-25

### Added
- JSON ベースのマクロデータ構造設計
- `macros_<ID>.json` フォーマット定義
- Book 40冊 × Set 10個 × Ctrl/Alt × 10スロット対応
- FFXIデータファイル（.dat/.ttl）読み込み機能

### Changed
- データアーキテクチャの確定

## [0.1.0] - 2025-10-15

### Added
- プロジェクト開始
- 基本的なプロジェクト構造
- PyQt6 ベースの UI フレームワーク選定
- FFXI マクロフォーマットの調査・解析

- **メジャーバージョン (1.x.x)**: 互換性のない大きな変更
- **マイナーバージョン (x.1.x)**: 後方互換性のある機能追加
- **パッチバージョン (x.x.1)**: 後方互換性のあるバグ修正
